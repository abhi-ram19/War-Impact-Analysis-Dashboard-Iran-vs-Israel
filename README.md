# 📊 War Impact Analysis Dashboard (Excel)

## 📌 Overview
This project presents an interactive Excel dashboard analyzing the socio-economic impact of war between Iran and Israel.

It focuses on key indicators such as unemployment, inflation, poverty, and war cost to provide a comparative analysis.

---

## 🔍 Key Insights

- 📉 Israel experienced a higher unemployment spike (~26%) compared to Iran (~18%)
- 📈 Iran faced higher inflation (~72%) indicating strong economic instability
- 🏚️ Poverty increased significantly in both countries, with Israel reaching ~34%
- 💰 Total war cost exceeded $3.4 trillion, showing massive economic burden
